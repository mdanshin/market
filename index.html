Кабинет продавца маркетплейса, каким он должен быть

Пульс продаж, аналитика за период, юнит-экономика заказа и география — в быстром нативном приложении. Ключи хранятся в системе, всё считается на вашем компьютере.

Скачать для macOS Все версии

Обзор и аналитика

Пульс дня и метрики за период со сравнением период-к-периоду: заказы, выручка, конверсия.

Финансы и юнит-экономика

Полная экономика заказа из Finance API: комиссии, логистика, эквайринг и чистая прибыль.

Заказы FBS и FBO

Отправления в одном списке со статусами ровно как в Ozon — без выдуманных названий и лишних кликов.

География заказов

Офлайн-карта России: откуда приходят заказы, по регионам — работает без интернета и без утечек.

Ключи API — в системном хранилище (Keychain). Запросы к маркетплейсу идут только из защищённого процесса, интерфейс никогда не видит секретов.
⚠️ Сборка без подписи Apple. При первом запуске: правый клик по приложению → «Открыть», затем подтвердите. Дальше — открывается как обычно. Подпись разработчика появится в следующих версиях.
